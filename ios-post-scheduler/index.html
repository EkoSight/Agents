📲 iPhone Standalone Guide

Since iOS blocks websites from scheduling alerts while closed, set up this 1-minute native automation:
1
Open the Shortcuts app on your iPhone and go to the Automation tab.
2
Tap Create Personal Automation (or +) and choose Time of Day.
3
Select Time of Day (e.g. daily at 09:00).
4
Add Action: Search for Safari and select Open URLs. Set the URL to this app's hosted link.
5
Turn OFF "Ask Before Running" to let it trigger silently. Now, your iPhone will wake up and prompt you precisely when the agent calculated!
